# Screenshots

Add your application screenshots here with the following filenames:

1. **home.png** - Home/landing page
2. **products.png** - Product listing page with search and pagination
3. **cart.png** - Shopping cart with items
4. **orders.png** - Order history page
5. **order-detail.png** - Individual order details/confirmation page

## Tips for Taking Screenshots

- Use browser's full-page screenshot feature or browser extension
- Make sure the application looks good (add sample products/orders)
- Include the navbar in screenshots to show navigation
- Use consistent browser window size for all screenshots
- Consider showing different states (empty cart, cart with items, etc.)
